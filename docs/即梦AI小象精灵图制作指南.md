## 第一步：创建角色设计稿

打开 [jimeng.ai](https://jimeng.ai)，选择 **"AI 图片生成"**

### 角色定义提示词（先出主图）

```
一只可爱的Q版小灰象，大耳朵，圆滚滚的身体，短短的四条腿，
小小的尾巴，黑色圆眼睛带高光，水彩手绘插画风格，
柔和的灰色配粉色耳朵内侧，简洁干净，白色纯色背景，
正面微侧站立，全身像，居中构图
```

**生成设置：**
- 图片比例：1:1
- 生成数量：4 张
- 选一张你觉得最满意、最可爱的作为 **角色基准图**

> **关键**：这张图决定了后续所有帧的风格，花时间挑好。
> 不满意就微调提示词重新生成，直到满意为止。

---

## 第二步：锁定角色一致性

在即梦中使用 **"参考图"** 功能：

1. 点击输入框旁的 **"参考图"** / **"图片参考"** 按钮
2. 上传你在第一步选好的角色基准图
3. **参考强度调到 0.7-0.8**（太高会死板，太低角色会变）

后续所有生成都 **保持这张参考图不变**。

## 第五步：统一尺寸 + 居中裁切

所有帧必须统一为 **200×200 像素**，角色居中。

用这个 Python 脚本批量处理（需要安装 `pip install Pillow`）：

```python
from PIL import Image
import os

def normalize_frame(input_path, output_path, size=200):
    """把任意尺寸的透明底图缩放+居中到 size×size"""
    img = Image.open(input_path).convert('RGBA')

    # 裁掉透明边缘
    bbox = img.getbbox()
    if bbox:
        img = img.crop(bbox)

    # 等比缩放，留 10% padding
    target = int(size * 0.9)
    ratio = min(target / img.width, target / img.height)
    new_w = int(img.width * ratio)
    new_h = int(img.height * ratio)
    img = img.resize((new_w, new_h), Image.LANCZOS)

    # 居中放到 200×200 画布
    canvas = Image.new('RGBA', (size, size), (0, 0, 0, 0))
    x = (size - new_w) // 2
    y = (size - new_h) // 2
    canvas.paste(img, (x, y), img)
    canvas.save(output_path)

# 用法：处理一个文件夹里所有 PNG
input_dir = './raw_frames/idle/'
output_dir = './frames/idle/'
os.makedirs(output_dir, exist_ok=True)

for f in sorted(os.listdir(input_dir)):
    if f.endswith('.png'):
        normalize_frame(
            os.path.join(input_dir, f),
            os.path.join(output_dir, f)
        )
        print(f'Done: {f}')
```

## 第六步：拼合 Sprite Sheet

把处理好的帧横向拼成一张图：

```python
from PIL import Image
import os

def make_sprite_sheet(frame_dir, output_path, frame_size=200):
    frames = sorted([
        f for f in os.listdir(frame_dir) if f.endswith('.png')
    ])
    if not frames:
        print(f'No frames in {frame_dir}')
        return

    sheet = Image.new('RGBA', (frame_size * len(frames), frame_size), (0, 0, 0, 0))
    for i, f in enumerate(frames):
        img = Image.open(os.path.join(frame_dir, f))
        sheet.paste(img, (i * frame_size, 0), img)

    os.makedirs(os.path.dirname(output_path), exist_ok=True)
    sheet.save(output_path)
    print(f'Sprite sheet saved: {output_path} ({len(frames)} frames, {frame_size * len(frames)}x{frame_size})')

# === 批量生成所有动作的 sprite sheet ===
actions = ['idle', 'happy', 'eating', 'hungry', 'excited', 'sad', 'reading', 'waving']

for action in actions:
    make_sprite_sheet(
        f'./frames/{action}/',
        f'./sprites/default/{action}.png'
    )
```

## 提示词调优技巧

- **角色不一致？** → 提高参考图强度到 0.85，或在提示词开头加"与参考图完全一致的角色"
- **风格偏差？** → 强调"水彩手绘插画风格"，加"柔和色调，简洁线条"
- **背景不干净？** → 加"纯白色背景，无任何装饰"
- **角色太小/太大？** → 加"全身像，角色占画面 80%，居中"
- **动作不明显？** → 把动作描述放在提示词最前面，加"动态姿势"

      ...
  frames/
    sailor-uniform/         ← 处理后 200×200
      idle/  01.png, 02.png
      ...
  sprites/
    sailor-uniform/         ← 最终 sprite sheet
      idle.png
      happy.png
      ...
```

最后复制到项目 `public/assets/pet/clothing/sailor-uniform/`

### 注意事项

- 每条提示词的**动作描述部分不用改**，只在统一前缀后加衣服描述
- 参考图用裸象对应帧（不是穿衣定义图），保证姿势对齐
- 参考强度 0.7 左右，太高衣服不明显，太低姿势偏差大
- 穿衣版小象与裸象的**体型轮廓应尽量一致**，这样切换时不会跳

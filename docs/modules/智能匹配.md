# 智能匹配模块

> 所属项目：Elephant Todo App  
> 最后更新：2026-02-26

---

## 一、功能说明

App 提供一个**统一输入入口**（底部导航"添加"按钮），用户随时输入内容（文字或语音），然后由系统智能匹配 + 用户选择决定内容去向。

### 交互流程

```
用户输入（文字 / 语音）
       │
       ▼
系统通过 LLM（DeepSeek/Gemini）智能匹配已有 Todo
       │
       ├── 找到相似 Todo → 提示用户选择：
       │       ├─ "关联到已有 Todo" → 作为随手记关联到该 Todo
       │       ├─ "新建 Todo"        → 创建为新的 Todo
       │       └─ "保存为随手记"    → 保存为独立的随手记
       │
       └── 未找到相似 → 提示用户选择：
               ├─ "新建 Todo"        → 创建为新的 Todo
               └─ "保存为随手记"    → 保存为独立的随手记
```

| 功能         | 说明                                       |
| ------------ | ------------------------------------------ |
| 统一输入     | 底部导航"添加"按钮唤起输入框，支持文字和语音 |
| 语音输入     | 长按麦克风按钮，语音识别自动转为文字。采用策略模式封装，默认使用 Web Speech API，后续可切换 |
| 智能匹配     | 输入后调用 DeepSeek/Gemini LLM，通过提示词模板与已有 Todo 进行语义匹配，返回相似 Todo 列表及分流建议 |
| 用户分流     | 用户根据匹配结果选择：关联已有 Todo / 新建 Todo / 保存为随手记 |

### 智能匹配技术方案

- **方案**：通过 DeepSeek/Gemini LLM 提示词模板进行语义匹配，返回相似 Todo 列表及分流建议
- **embedding 字段**：todos 表和 ideas 表中的 `embedding` 字段为预留字段，当前智能匹配由 LLM 直接完成
- **认证**：通过 API Key 认证，Key 存储在服务端环境变量中

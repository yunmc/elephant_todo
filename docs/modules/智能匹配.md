# 智能匹配模块

> 所属项目：Elephant Todo App  
> 最后更新：2026-02-26

---

## 一、功能说明

App 提供一个**统一输入入口**（底部导航"添加"按钮），用户随时输入内容（文字或语音），然后由系统智能匹配 + 用户选择决定内容去向。

### 交互流程

```
用户输入（文字 / 语音）
       │
       ▼
系统通过 LLM（DeepSeek/Gemini）查找相似的已有 Todo
       │
       ├── 找到相似 Todo → 显示列表，用户点击可关联
       │
       └── 未找到相似 → 用户可选择：
               ├─ "新建 Todo"        → 创建为新的 Todo
               └─ "保存为随手记"    → 保存为独立的随手记
```

| 功能         | 说明                                       |
| ------------ | ------------------------------------------ |
| 统一输入     | 底部导航"添加"按钮唤起输入框，支持文字和语音 |
| 语音输入     | 长按麦克风按钮，语音识别自动转为文字（Web Speech API），15 秒超时自动停止 |
| 相似匹配     | 输入后调用 DeepSeek/Gemini LLM，查找与已有 Todo 语义相似的条目（最多 3 个） |
| 用户操作     | 用户可选择：点击关联已有 Todo / 新建 Todo / 保存为随手记 |

### 智能匹配技术方案

- **方案**：通过 DeepSeek/Gemini LLM 提示词模板进行语义匹配，返回相似 Todo 列表
- **Provider 策略**：优先调用已配置 Key 的 provider，失败时自动回退备用 provider（30s 超时）
- **embedding 字段**：todos 表和 ideas 表中的 `embedding` 字段为预留字段，当前智能匹配由 LLM 直接完成
- **认证**：通过 API Key 认证，Key 存储在服务端环境变量中
- **限流**：20 次/5 分钟（按 IP）
- **输入限制**：最多 500 字

# 用户与认证模块

> 所属项目：Elephant Todo App  
> 最后更新：2026-02-26

---

## 一、功能说明

| 功能     | 说明                                       |
| -------- | ------------------------------------------ |
| 注册     | 用户名 + 邮箱 + 密码注册                  |
| 登录     | 邮箱 + 密码登录，返回 JWT Token           |
| 退出     | 清除本地 Token                             |
| 忘记密码 | 输入注册邮箱 → 发送重置链接/验证码到邮箱 → 设置新密码 |
| 修改密码 | 登录状态下，输入旧密码 + 新密码进行修改           |
| 认证保持 | Token 自动刷新机制，支持多设备同时登录     |

---

## 二、API 接口

| 方法 | 路径              | 说明     | 认证 |
| ---- | ----------------- | -------- | ---- |
| POST | /api/auth/register        | 用户注册 | 否   |
| POST | /api/auth/login           | 用户登录 | 否   |
| POST | /api/auth/forgot-password | 发送密码重置邮件（输入邮箱） | 否 |
| POST | /api/auth/reset-password  | 重置密码（携带重置 Token + 新密码） | 否 |
| POST | /api/auth/change-password  | 修改密码（输入旧密码 + 新密码） | 是 |
| POST | /api/auth/refresh         | 刷新 Token | 是 |
| GET  | /api/auth/me              | 获取当前用户信息 | 是 |

## 三、数据库表

### 3.1 用户表（users）

| 字段       | 类型         | 约束                    | 说明               |
| ---------- | ------------ | ----------------------- | ------------------ |
| id         | INT          | PRIMARY KEY, AUTO_INCREMENT | 主键，自增     |
| username   | VARCHAR(50)  | NOT NULL, UNIQUE        | 用户名，唯一       |
| email      | VARCHAR(100) | NOT NULL, UNIQUE        | 邮箱，唯一         |
| password   | VARCHAR(255) | NOT NULL                | 密码（bcrypt 加密）|
| vault_salt | VARCHAR(50)  | DEFAULT NULL            | 密码本加密盐值（per-user random salt） |
| created_at | DATETIME     | NOT NULL, DEFAULT NOW() | 创建时间           |
| updated_at | DATETIME     | NOT NULL, DEFAULT NOW() ON UPDATE NOW() | 更新时间 |

**索引**：
- `uk_users_username` — UNIQUE(username)
- `uk_users_email` — UNIQUE(email)

### 3.2 密码重置令牌表（password_reset_tokens）

| 字段       | 类型         | 约束                    | 说明               |
| ---------- | ------------ | ----------------------- | ------------------ |
| id         | INT          | PRIMARY KEY, AUTO_INCREMENT | 主键，自增     |
| user_id    | INT          | NOT NULL, FOREIGN KEY   | 外键 → users.id    |
| token      | VARCHAR(255) | NOT NULL, UNIQUE        | 重置令牌（随机字符串） |
| expires_at | DATETIME     | NOT NULL                | 过期时间           |
| used       | BOOLEAN      | NOT NULL, DEFAULT FALSE | 是否已使用         |
| created_at | DATETIME     | NOT NULL, DEFAULT NOW() | 创建时间           |

**索引**：
- `uk_reset_tokens_token` — UNIQUE(token)
- `idx_reset_tokens_user_id` — INDEX(user_id)

**外键**：
- `fk_reset_tokens_user` — user_id → users(id) ON DELETE CASCADE

## 四、建表 SQL

```sql
CREATE TABLE users (
  id         INT          AUTO_INCREMENT PRIMARY KEY,
  username   VARCHAR(50)  NOT NULL,
  email      VARCHAR(100) NOT NULL,
  password   VARCHAR(255) NOT NULL,
  vault_salt VARCHAR(50)  DEFAULT NULL,
  created_at DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  UNIQUE KEY uk_users_username (username),
  UNIQUE KEY uk_users_email (email)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE password_reset_tokens (
  id         INT          AUTO_INCREMENT PRIMARY KEY,
  user_id    INT          NOT NULL,
  token      VARCHAR(255) NOT NULL,
  expires_at DATETIME     NOT NULL,
  used       BOOLEAN      NOT NULL DEFAULT FALSE,
  created_at DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uk_reset_tokens_token (token),
  KEY idx_reset_tokens_user_id (user_id),
  CONSTRAINT fk_reset_tokens_user FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

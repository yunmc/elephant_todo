# Todo 管理 — 测试用例

> 所属项目：Elephant Todo App  
> 测试框架：Vitest  
> 最后更新：2026-02-26

---

## 一、单元测试 / API 测试

### 1.1 Todo CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| TODO-C01 | 创建 Todo（标题 + 描述 + 优先级 + 截止日期） | 201，返回新 Todo |
| TODO-C02 | 仅标题创建 Todo | 201，其他字段使用默认值 |
| TODO-C03 | 缺少标题 | 400，参数校验失败 |
| TODO-R01 | 获取 Todo 列表 | 200，返回列表 + 分页信息 |
| TODO-R02 | 获取单个 Todo 详情 | 200，返回完整 Todo 信息 |
| TODO-R03 | 获取不存在的 Todo | 404 |
| TODO-R04 | 获取其他用户的 Todo | 404（权限隔离） |
| TODO-U01 | 更新 Todo 标题/描述 | 200，返回更新后的 Todo |
| TODO-U02 | 更新不存在的 Todo | 404 |
| TODO-D01 | 删除 Todo | 200，删除成功 |
| TODO-D02 | 删除不存在的 Todo | 404 |
| TODO-D03 | 删除其他用户的 Todo | 404 |

### 1.2 完成/恢复

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| TODO-T01 | 切换未完成 → 完成 | 200，status='completed'，completed_at 非空 |
| TODO-T02 | 切换完成 → 未完成 | 200，status='pending'，completed_at 为空 |

### 1.3 筛选与搜索

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| TODO-F01 | 按状态筛选（pending） | 仅返回未完成 Todo |
| TODO-F02 | 按状态筛选（completed） | 仅返回已完成 Todo |
| TODO-F03 | 按优先级筛选（high） | 仅返回高优先级 Todo |
| TODO-F04 | 按关键词搜索 | 返回标题/描述中包含关键词的 Todo |
| TODO-F05 | 按分类筛选 | 仅返回指定分类的 Todo |
| TODO-F06 | 按日期范围筛选 | 返回截止日期在范围内的 Todo |
| TODO-F07 | 分页（page=2, limit=10） | 返回第二页数据 + 正确的分页信息 |

### 1.4 子任务

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| SUB-C01  | 为 Todo 创建子任务 | 201，返回子任务 |
| SUB-C02  | 父 Todo 不存在时创建子任务 | 404 |
| SUB-R01  | 获取 Todo 的子任务列表 | 200，按 sort_order 排序 |
| SUB-U01  | 更新子任务标题 | 200 |
| SUB-T01  | 切换子任务完成状态 | 200 |
| SUB-D01  | 删除子任务 | 200 |
| SUB-D02  | 删除 Todo 时级联删除子任务 | 子任务被自动删除 |

### 1.5 关联随手记

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| TODO-I01 | 获取 Todo 关联的 Idea 列表 | 200，返回关联的 Idea 列表 |
| TODO-I02 | Todo 无关联 Idea | 200，返回空列表 |

## 一、单元测试 / API 测试

### 1.1 CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| ID-C01 | 创建重要日期（标题 + 日期） | 201 |
| ID-C02 | 创建时指定图标、备注、农历、每年重复 | 201，所有字段正确保存 |
| ID-C03 | 缺少标题 | 400 |
| ID-C04 | 缺少日期 | 400 |
| ID-R01 | 获取列表 | 200，含 days_until 字段，按最近排序 |
| ID-U01 | 更新标题/日期 | 200 |
| ID-D01 | 删除日期 | 200 |

### 1.2 倒计时计算

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| ID-T01 | 未来日期 | days_until > 0 |
| ID-T02 | 今天的日期 | days_until = 0 |
| ID-T03 | 过去日期（非重复） | days_until < 0 |
| ID-T04 | 每年重复 + 今年已过 | days_until 计算到明年的同一天 |
| ID-T05 | 每年重复 + 今年未到 | days_until 计算到今年的同一天 |

### 1.3 排序

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| ID-S01 | 多个日期列表 | 按 days_until 升序排列，最近的排前面 |

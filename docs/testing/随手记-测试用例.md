# 随手记 — 测试用例

> 所属项目：Elephant Todo App  
> 测试框架：Vitest  
> 最后更新：2026-02-26

---

## 一、单元测试 / API 测试

### 1.1 Idea CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| IDEA-C01 | 创建随手记（文字来源） | 201，source='text' |
| IDEA-C02 | 创建随手记（语音来源） | 201，source='voice' |
| IDEA-C03 | 缺少内容 | 400 |
| IDEA-R01 | 获取随手记列表 | 200，按时间倒序 + 分页 |
| IDEA-R02 | 获取单个随手记 | 200 |
| IDEA-R03 | 获取不存在的随手记 | 404 |
| IDEA-U01 | 更新随手记内容 | 200 |
| IDEA-D01 | 删除随手记 | 200 |

### 1.2 关联 Todo

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| IDEA-L01 | 将随手记关联到 Todo | 200，todo_id 更新 |
| IDEA-L02 | 关联到不存在的 Todo | 404 |
| IDEA-L03 | 取消关联 | 200，todo_id 置空 |
| IDEA-L04 | 对已关联的随手记再次关联新 Todo | 200，更新 todo_id |

### 1.3 转化为 Todo

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| IDEA-V01 | 将未关联的随手记转化为 Todo | 200，创建新 Todo，随手记关联到该 Todo |
| IDEA-V02 | 将已关联的随手记转化 | 400/409（已关联不可转化） |

### 1.4 筛选

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| IDEA-F01 | 按关联状态筛选（已关联） | 仅返回 todo_id 非空的随手记 |
| IDEA-F02 | 按关联状态筛选（未关联） | 仅返回 todo_id 为空的随手记 |
| IDEA-F03 | 按来源筛选（text） | 仅返回文字来源 |
| IDEA-F04 | 按来源筛选（voice） | 仅返回语音来源 |

| 编号 | 用例 | 预期结果 |

# 分类与标签 — 测试用例

> 所属项目：Elephant Todo App  
> 测试框架：Vitest  
> 最后更新：2026-02-26

---

## 一、单元测试 / API 测试

### 1.1 分类 CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| CAT-C01 | 创建分类（名称 + 颜色） | 201 |
| CAT-C02 | 创建同名分类 | 400/409，唯一性约束 |
| CAT-C03 | 缺少名称 | 400 |
| CAT-R01 | 获取分类列表 | 200，仅返回当前用户的分类 |
| CAT-U01 | 更新分类名称 | 200 |
| CAT-U02 | 更新为已存在的名称 | 400/409 |
| CAT-D01 | 删除分类 | 200 |
| CAT-D02 | 删除已关联 Todo 的分类 | 200，Todo 的 category_id 置空（SET NULL） |

### 1.2 标签 CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| TAG-C01 | 创建标签 | 201 |
| TAG-C02 | 创建同名标签 | 400/409，唯一性约束 |
| TAG-R01 | 获取标签列表 | 200，仅返回当前用户的标签 |
| TAG-U01 | 更新标签名称 | 200 |
| TAG-D01 | 删除标签 | 200 |
| TAG-D02 | 删除已关联 Todo 的标签 | 200，todo_tags 关联被级联删除 |

## 二、级联行为测试

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| CAT-K01 | 删除分类 → 关联 Todo | Todo 保留，category_id 置 NULL |
| TAG-K01 | 删除标签 → todo_tags 关联 | 关联记录被级联删除 |
| TAG-K02 | 删除 Todo → todo_tags 关联 | 关联记录被级联删除 |

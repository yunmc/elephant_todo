# 记账 — 测试用例

## 一、单元测试 / API 测试

### 1.1 记账分类 CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| FC-C01 | 创建支出分类（名称 + 图标 + type=expense） | 201 |
| FC-C02 | 创建收入分类（type=income） | 201 |
| FC-C03 | 创建同名同类型分类 | 400/409，唯一性约束 |
| FC-C04 | 不同类型下同名分类 | 201（允许，expense 和 income 下可同名） |
| FC-C05 | 缺少名称 | 400 |
| FC-R01 | 获取分类列表 | 200，按 sort_order 排序 |
| FC-U01 | 更新分类名称/图标 | 200 |
| FC-D01 | 删除分类 | 200 |
| FC-D02 | 删除有记录的分类 | 200，记录的 category_id 置空 |

### 1.2 记账记录 CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| FR-C01 | 创建支出记录（金额 + 分类 + 日期） | 201 |
| FR-C02 | 创建收入记录 | 201 |
| FR-C03 | 缺少金额 | 400 |
| FR-C04 | 缺少类型（income/expense） | 400 |
| FR-C05 | 缺少日期 | 400 |
| FR-C06 | 金额为 0 或负数 | 400 |
| FR-R01 | 获取记录列表 | 200，含分类信息 + 分页 |
| FR-U01 | 更新记录金额/备注 | 200 |
| FR-D01 | 删除记录 | 200 |

### 1.3 筛选

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| FR-F01 | 按类型筛选（expense） | 仅返回支出记录 |
| FR-F02 | 按类型筛选（income） | 仅返回收入记录 |
| FR-F03 | 按分类筛选 | 仅返回指定分类的记录 |
| FR-F04 | 按日期范围筛选 | 返回指定日期范围内的记录 |
| FR-F05 | 分页查询 | 返回正确的分页数据和分页信息 |

### 1.4 统计

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| FS-01 | 获取当月统计 | 200，返回 total_income / total_expense / balance |
| FS-02 | 获取指定月份统计 | 200，返回该月数据 |
| FS-03 | 按分类汇总 | 200，by_category 包含每个分类的金额 |
| FS-04 | 无记录的月份 | 200，总收入/支出/结余均为 0 |

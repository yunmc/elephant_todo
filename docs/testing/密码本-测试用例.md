# 密码本 — 测试用例

> 所属项目：Elephant Todo App  
> 测试框架：Vitest  
> 最后更新：2026-02-26

---

## 一、单元测试 / API 测试

### 1.1 密码分组 CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| VG-C01 | 创建分组 | 201 |
| VG-C02 | 创建同名分组 | 400/409，唯一性约束 |
| VG-R01 | 获取分组列表 | 200，按 sort_order 排序 |
| VG-U01 | 更新分组名称 | 200 |
| VG-D01 | 删除分组 | 200 |
| VG-D02 | 删除有密码记录的分组 | 200，密码记录的 group_id 置空 |

### 1.2 密码记录 CRUD

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| VE-C01 | 创建密码记录（含加密数据） | 201 |
| VE-C02 | 缺少名称 | 400 |
| VE-C03 | 缺少 encrypted_data | 400 |
| VE-R01 | 获取密码列表 | 200，含分组信息 |
| VE-R02 | 按分组筛选 | 仅返回指定分组的记录 |
| VE-R03 | 按名称搜索 | 返回名称匹配的记录 |
| VE-R04 | 获取单个密码记录 | 200，包含 encrypted_data |
| VE-U01 | 更新密码记录 | 200 |
| VE-D01 | 删除密码记录 | 200 |

### 1.3 批量更新（修改密码后重新加密）

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| VE-B01 | 批量更新所有记录的 encrypted_data | 200，所有记录更新成功 |
| VE-B02 | 批量更新中某条失败 | 事务回滚，所有记录保持原状 |

### 1.4 加密安全性

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| VE-S01 | API 响应中不含明文密码 | encrypted_data 为密文 |
| VE-S02 | 数据库中不存储明文 | encrypted_data 列为 Base64 编码密文 |

## 二、加密逻辑测试（客户端）

| 编号 | 用例 | 预期结果 |
| ---- | ---- | -------- |
| CRYPTO-01 | 加密后解密还原 | 数据一致 |
| CRYPTO-02 | 不同密码派生不同密钥 | 无法用另一个密码解密 |
| CRYPTO-03 | 修改密码后重新加密 | 旧密钥解密 → 新密钥加密，数据不丢失 |
| CRYPTO-04 | 每条记录使用不同 IV | 同内容不同密文 |
